# 🛡️ Insurance Premium Prediction using Machine Learning

## 📌 Project Overview
This project predicts the estimated insurance premium based on user details using a Machine Learning model. The model was trained in Jupyter Notebook and deployed using Streamlit.

## 🚀 Features
- Predicts insurance premium instantly
- Simple and user-friendly web interface
- Fast and accurate predictions
- Machine Learning model integrated with Streamlit

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib
- Streamlit

## 📂 Project Structure

```
InsurancePremiumPrediction/
│── app.py
│── insurance_model.pkl
│── insurance_prediction.ipynb
│── insurance.csv
│── requirements.txt
│── README.md
```

## 📊 Input Features
- Age
- Gender
- BMI
- Number of Children
- Smoker Status
- Region

## 💰 Output
The application predicts the estimated insurance premium based on the user's information.

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📦 Model
The Machine Learning model was trained using the Insurance Premium dataset and saved as `insurance_model.pkl`.
